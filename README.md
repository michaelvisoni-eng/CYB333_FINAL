# CYB333 Final Project

## Overview
This project is a Python-based port scanner developed for CYB333 Security Automation that will execute a TCP port scan to your desired target in desired port range and generate a .txt and .json file with the results for parsing. **Only scan systems you own or have permission to test.**

## Features
- TCP port scanning
- User-defined target host
- User-defined port range
- Open port detection
- Scan results reporting

## Requirements
- Python 3.x

## Download
sudo apt update
sudo apt install git
git clone https://github.com/michaelvisoni-eng/CYB333_FINAL.git

## Usage
Run the script:

python3 portscanner2.py
Enter the target IP address and port range when prompted. 
Scan results will be automatically be saved to directory as a .txt and .json file.
